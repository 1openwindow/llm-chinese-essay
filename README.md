# 大语言模型中文论文

随着大语言模型的发展，其在各个领域的应用越来越广泛。然而，大部分的研究成果都是以英文论文的形式发布，这对非英语母语的读者来说，阅读起来有一定的困难。因此，这个项目希望通过收集和翻译这些论文，让更多的人能够了解和学习大语言模型的最新研究进展。
